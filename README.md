# testing-js

A quick journey through the basics of js testing

## How To

Go to a lesson branch `lessonN` to see the new lesson, go to `master` for the full course

Set up the project:
```bash
yarn install
```

Launch tests and watch changes:
```bash
yarn test
```

Launch tests with coverage:
```bash
yarn coverage
```
